# Jarvis-AI
Transformer-powered AI voice assistant with wake word, context memory, and offline capabilities — inspired by JARVIS.
Features offline voice interaction, wake-word detection (“Hey Jarvis”), contextual memory, intent confidence handling, and modular architecture. Designed step-by-step to demonstrate how real-world voice assistants are engineered, from speech recognition to intelligent action execution.

TechStack Used:
-->Python

-->PyTorch

-->Hugging Face Transformers (DistilBERT)

-->SpeechRecognition

-->Pyttsx3 (TTS)

-->Offline / Online Wake Word Support
